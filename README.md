# stock_database_v1

# P4阶段完成：股票数据查询与分析系统

## 🎯 新增功能概览

### 1. 数据查询API (`src/interfaces/query_api.py`)
- **股票基本信息查询**：获取所有股票、指数成分股
- **价格数据查询**：日线数据查询（支持日期范围）
- **统计分析**：价格摘要、相关性矩阵
- **实时查询**：最新价格、数据验证

### 2. 技术指标计算器 (`src/calculators/technical_indicators.py`)
- **趋势指标**：SMA, EMA, WMA
- **动量指标**：RSI, MACD, 随机指标
- **波动率指标**：布林带, ATR
- **成交量指标**：OBV, MFI, 成交量比率
- **综合指标**：所有技术指标一键计算
- **交易信号**：自动生成买卖信号
- **绩效评估**：夏普比率、最大回撤等

### 3. 股票分析API (`src/interfaces/analysis_api.py`)
- **单股票分析**：技术分析、风险分析、综合评估
- **多股票比较**：表现对比、排名分析
- **投资组合分析**：组合绩效、风险分解、优化建议
- **报告生成**：文本、JSON格式分析报告
